\subsection{Detailed Description}
This class defines the N\+O\+R\+M\+AL state of the \hyperlink{namespaceFSM}{F\+SM}. 

At each execution it randomly chooses how many movements the robot will make and put this number in the counter variable. Then there is a while loop which checks\+:
\begin{DoxyItemize}
\item If there is a \char`\"{}play\char`\"{} message, then the \hyperlink{namespaceFSM}{F\+SM} must go in P\+L\+AY state
\item If it has been iterated as many times as the counter variable then it goes in S\+L\+E\+EP state Otherwise it make a rquest to the Navigation node to go in X and Y position 
\end{DoxyItemize}
