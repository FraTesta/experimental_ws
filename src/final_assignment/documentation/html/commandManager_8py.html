Detailed Description

This script is a node which is the core of the entier project.

This node includes the subsription to State and GetPosition publishers, And implement a finite state machine which manages the information coming from the two publisher and changes the state of the system in according to them.

And implement a finite state machine with five states which are described in the README file. It manages the messages comming from other nodes and ROS packeges.
